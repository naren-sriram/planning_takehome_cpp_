# Visualizer Controls - Quick Reference

## Layout

```
┌─────────────────────────────────────────┐
│     Density Map & Path Animation        │
│  (Blue robot moving on colored grid)    │
└─────────────────────────────────────────┘
         ┌─────────┬─────────┐
         │  Pause  │ Replay  │
         └─────────┴─────────┘
```

## Two Simple Buttons

### 1. Pause/Play Button (Left)
- **When Red**: Animation is playing
- **When Green**: Animation is paused

**How to use**:
- Click → Pause (button turns green)
- Click again → Resume from same spot (button turns red)

### 2. Replay Button (Right)
- **Always Blue**

**How to use**:
- Click → Restart from beginning
- Robot returns to origin dock
- Path clears and replays

## Common Actions

**Want to study a specific moment?**
1. Click Pause when robot reaches that spot
2. Examine the map
3. Click Play to continue

**Want to watch again?**
1. Click Replay
2. Animation starts from the beginning

**Want to see next order?**
- Close the window

## That's It!
Simple, clean controls for effective visualization.
